## ğŸ§© Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Slash-495/chat-app
cd chat-app
```

### 2. Setup Backend

```bash
cd backend
npm install
npm start
```

> Make sure to set up your `.env` file for the backend with:
```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```

### 3. Setup Frontend

```bash
cd ../frontend
npm install
npm run dev
```
